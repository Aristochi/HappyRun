# HappyRun
这是一款轻量、简易的、采用高德地图SDK记录轨迹和三轴加速度传感器的跑步、计步Android APP应用作业
## 简要功能介绍
**跑步模块**
通过高德地图SDK记录每一次跑步的轨迹，并将每一天跑步的里程、时间、记录在数据库里，支持查看历史跑步记录。

**计步模块**
利用三轴加速度传感器，记录一天走过的步数、允许设置每天的锻炼计划，以及提供历史记录起到监督反省自己的作用
